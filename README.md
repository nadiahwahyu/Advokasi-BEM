# Sistem Advokasi BEM

Sistem Advokasi BEM adalah platform berbasis web untuk mengelola layanan advokasi oleh Badan Eksekutif Mahasiswa (BEM). Sistem ini mempermudah mahasiswa untuk mengajukan laporan, memantau status advokasi, dan berkomunikasi dengan tim BEM secara terstruktur dan aman.

## Fitur Utama
- Pengajuan laporan advokasi oleh mahasiswa
- Monitoring status laporan
- Manajemen data advokasi oleh tim BEM
- Autentikasi berbasis peran (role-based access)
- Integrasi dengan sistem notifikasi

## Teknologi yang Digunakan
- **Backend:** PHP, MySQL
- **Frontend:** HTML, CSS, JavaScript
- **Versi Kontrol:** Git, GitHub
